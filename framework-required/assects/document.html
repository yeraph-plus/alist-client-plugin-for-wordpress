WP Settings Framework Function Document


building up a framework page：


AYF::new_opt(array(
    'title' => 'AIYA-CMS',
    'slug' => 'option_page_name',
    'desc' => 'AIYA-CMS Theme',
    'fields' => array(
        array(
           'title' => 'field name',
            'desc' => 'this is a field demo',
            'id'   => 'field_demo',
            'type' => 'switch',
            'default' => false,
        ),
        array(
           'type' => 'content',
            'desc' => 'this is a field demo',
        ),
    )
));

how to get settings:


get the swtich (will verify and return a boolean data):

AYF::get_checked('field_demo', 'option_page_name');


else , only return settings content:

AYF::get_opt('field_demo', 'option_page_name');
